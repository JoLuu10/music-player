
# Music Player

Un pequeño reproductor de música creado en React con canciones de Eternal Sunshin de Ariana Grande, con barra de progreso, volumen y botones de play, pause y next.


## Creación de un archivo en React
Paso 1. Creamos el archivo 
```bash
  npx create-react-app music-player
```
Paso 2. Entramos en la carpeta
```bash
  cd music-player
```
Paso 3. Iniciamos
```bash
  npm start
```
